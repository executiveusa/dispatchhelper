# Spy-Themed Photo Assets

This directory contains all the spy-themed visual assets for the landing page, styled after the 007 SpyScape protocol.

## Directory Structure

### `/hero`
- `hero-background.svg` - Full-page hero background with dark spy aesthetic, tech grid overlay, and vehicle silhouette

### `/vehicles`
- `spy-vehicle-1.svg` - Sleek dispatch vehicle styled as spy car (sedan profile) - "DISPATCH UNIT ALPHA"
- `spy-vehicle-2.svg` - SUV-style dispatch vehicle styled as spy vehicle - "DISPATCH UNIT BRAVO"

### `/gadgets`
- `communication-device.svg` - AI Dispatch System represented as a spy communication device
- `tracking-system.svg` - Fleet tracking system shown as GPS/radar tracking display
- `intelligence-hub.svg` - AI neural core represented as intelligence processing hub

### `/agents`
- `agent-profile.svg` - Dispatcher profile shown as agent silhouette with security badge

### `/backgrounds`
- `command-center.svg` - Dispatch command center styled as spy operations headquarters with multiple monitors

## Theme Guidelines

All assets follow the spy-themed color palette:
- **Primary Dark**: #0a0a0a, #0f172a, #1a1a2e
- **Accent Blue**: #3b82f6, #60a5fa, #1e3a8a  
- **Tech Gray**: #334155, #475569, #64748b
- **Status Green**: #22c55e
- **Alert Yellow/Red**: #eab308, #ef4444

## Usage

These assets are designed to work together to create an immersive spy/007-themed experience while maintaining the dispatch functionality narrative.

### Dispatch → Spy Terminology Mapping:
- Dispatch Operations → Mission Control
- Drivers → Field Agents
- Fleet Tracking → Agent Tracking
- AI Assistant → Intelligence System
- Command Center → Operations HQ
- Dispatch Vehicles → Mission Units
